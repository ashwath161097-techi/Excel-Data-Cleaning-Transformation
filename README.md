# Excel Data Cleaning & Transformation Project

## 📊 Overview
This project focuses on cleaning and transforming raw data into a structured and usable format using Microsoft Excel.

## 📁 Dataset
- Excel- Data Cleaning and Transformation.xlsx

## 🎯 Objectives
- Handle missing values
- Remove duplicates
- Standardize data formats
- Transform raw data into structured format

## 🛠 Tools Used
- Microsoft Excel

## 🔧 Techniques Applied
- Data cleaning
- Text to columns
- Find & Replace
- Removing duplicates
- Formatting and standardization

## 📈 Outcome
- Clean, structured dataset ready for analysis

## 🚀 How to Use
1. Download the dataset
2. Open in Excel
3. Review cleaned and transformed data
